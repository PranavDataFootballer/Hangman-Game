# Hangman-Game
This project is a fun and interactive Hangman Game built entirely using Python. The player must guess the hidden word one letter at a time before running out of lives.  The game includes:  

Random word generation 
ASCII art visuals 
Live game progress updates 
Win/Loss detection 
Repeated guess handling 
Terminal-based interactive gameplay

Features:-

Interactive Terminal Gameplay
The game continuously updates the hidden word as the player guesses letters.

Random Word Selection
Every game is different thanks to Python's random.choice().

ASCII Hangman Art
Visual stages make the game more engaging and fun.

Duplicate Guess Detection
The game warns users if they repeat the same guess.

Win & Lose Conditions
The game automatically detects:

Victory
Game Over
